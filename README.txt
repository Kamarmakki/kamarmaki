== Description ==
Kamar SEO is a lightweight, modern, and fully Arabic WordPress theme designed for SEO agencies and digital marketing services. Fully RTL, fast, and minimal.

== Features ==
- Fully Arabic & RTL
- Clean and modern design
- Shortcode: [kamar_contact_form]
- Ajax contact form
- Lightweight CSS & JS
- WordPress coding standards

== Installation ==
1. Upload the theme folder to `/wp-content/themes/`
2. Activate it from Appearance > Themes
3. Use shortcode `[kamar_contact_form]` in any page

== Frequently Asked Questions ==
= Can I use this theme for non-SEO websites? =
Yes, it's flexible.

= Is it translation-ready? =
Yes, text domain: `kamar-seo`
